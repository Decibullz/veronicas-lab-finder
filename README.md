# Veronica's Lab Finder

An application used to search correct test tube colors for required lab draws. User can Create, Edit or Delete labs as needed from the view all panel. 

## Tech Used

* HTML
* CSS3
* JavaScript
* Node.JS
* Express.JS
* Mongoose.JS
* MongoDB
* Morgan
* EJS
* Google Fonts
* Heroku
